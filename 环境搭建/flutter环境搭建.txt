搭建JAVA环境：
Windows x86 32位系统
Windows x64 64位系统，是兼容32位系统的。
从  https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html  下载jdk，解压后配置环境变量。  可以输入 java 和 java -version 来检测是否搭建成功。
搭建flutter环境：
从  https://flutter.dev/docs/development/tools/sdk/releases#windows  下载jdk，解压后配置环境变量。  可以输入 flutter doctor 来检测是否搭建成功。
搭建安卓环境：
1、从  https://developer.android.google.cn/   下载Android Studio，搭建安卓sdk和开发环境。
2、打开Android Stuido 软件，然后找到Plugin的配置，搜索Flutter插件,然后点击安装。安装完成后，你需要重新启动一下Android Studio软件。
3、运行flutter doctor --android-licenses 来安装证书。
4、Android Stuido新建一个flutter工程，启动模拟器，点击运行。

VSCode下如何玩转Flutter：
1、从  https://code.visualstudio.com/  下载安装VSCode
2、在VSCode里面下载安装flutter插件
3、需要Android Stuido新建一个flutter工程。
4、启动模拟器或者手机：
	新建.bat文件手动启动  路径跟安卓sdk路径有关，内容：
E:\android\sdk\tools\emulator.exe -netdelay none -netspeed full -avd AndroidTV
如果模拟器名字有空格，用下划线替换。
	命令启动有问题  emulator -netdelay none -netspeed full -avd AndroidTV
5、在VSCode的命令窗（ctrl+~）执行 flutter run 即可。
6、按 r 显示热更新，按p显示网格，再按p不显示网格,按q退出，按o切换安卓和ios系统。
